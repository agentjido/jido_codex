# JidoCodex

OpenAI Codex CLI adapter for [JidoHarness](https://github.com/agentjido/jido_harness).

## Installation

Add `jido_codex` and `jido_harness` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:jido_harness, "~> 0.1"},
    {:jido_codex, "~> 0.1"}
  ]
end
```
